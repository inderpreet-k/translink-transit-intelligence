# TransLink Transit Intelligence Pipeline

**Technical Lead** · INFO 4190 Integration Project I · Kwantlen Polytechnic University · Spring–Summer 2026

A GTFS-powered commuter intelligence dashboard for TransLink (Metro Vancouver) — transforms raw scheduled transit data into plain-language route recommendations and interactive network visualizations.

---

## Overview

**242 routes · 63,250 weekly trips** (May 2026 GTFS snapshot)

Four dashboard views: Network Overview · Route Analyser · Stop Deep-Dive · Comparison Tool

Stack: PostgreSQL · Flask · Chart.js · Pandas · K-Means clustering (stretch feature)
